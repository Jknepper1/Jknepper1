### The Aphasiafier

AI-driven data pipline designed to be used on bulk transcripts for persons with Aphasia or other speech disorders. Designed to function as a CLI tool with the capability to handl JSON config file input. The data is processed through TTS, realtime audio, and transcription AI models in order to produce data for research purposes. Iterations are taken with the data going throguh this pipleine in order to develop prompts for Artificial Intelligence models that most accuratly match Natural Language features of real transcripts from real people with Aphasia.

This python was originally developed with NodeJS but I switched to Python as the amount of data processing and organization increased throughout the course of the research process. I ported the code due to the simpler nature of Python scripting and extensive library support for working with audio files, large datasets, and OpenAI's API.
